# App-Integration — Application Web d'Intégration de Données

Application web développée lors de mon stage PFE chez **Groupe ARKEOS** 
(Casablanca, Maroc) — Avril 2022.

## Description
Plateforme web permettant l'intégration et la gestion de données 
provenant de sources multiples, avec une interface d'administration moderne.

## Technologies utilisées
- **Backend** : Django (Python)
- **Frontend** : HTML, CSS, SCSS, JavaScript, Bootstrap
- **Base de données** : SQLite

## Comment lancer le projet
```bash
git clone https://github.com/ELHEBABY/App-Integration
cd App-Integration
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
